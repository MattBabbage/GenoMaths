# GenoMaths
Basis for a Genetic Algorithm to generate mathematical equations

What does this algorithm do?
Step 1:
Generates a population of linked trees, each of which can be simplified into a mathematical equation
Step 2:
Completes standard genetic processes Mutation, Crossover...
Step 3:
Check each individuals fitness against whatever dataset you choose to use
Step 4:
Selection based on fittest individuals, elitism ect.
Step 5:
Go back to step 2.

This repeats until you have generated an equation which matches your dataset. 
It works in N Dimensions aswell!
